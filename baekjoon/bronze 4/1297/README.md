# Math클래스

## 거듭제곱 구하기

### Math.pow()메서드의 입력값과 출력값은 모두 double형

```java
    double result = Math.pow(5, 2); //5의제곱
    System.out.println("5의 제곱은 : "+result);
    // 25 출력
```

## 제곱근(루트) 구하기

### Math.pow()메서드의 입력값과 출력값은 모두 double형

```java
    double result = Math.sqrt(25); //25의 제곱근
    System.out.println("25의 제곱근 : "+ result);
    // 5 출력
```
