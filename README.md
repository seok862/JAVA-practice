# Eclipse 단축키

- 단축키 전체 목록 : ctrl + shift + L <br>
- 글자 크기 조절 : ctrl + +,- <br>
- 한 줄 삭제 : ctrl + D <br>
- 행 단위 복사 : ctrl + alt + down <br>
- 멀티 컬럼 편집 : alt + shift + A <br>
- 행단위 이동 : alt + up, down <br>
- 자동 들여쓰기 : ctrl + i 
- 주석처리(한 줄) : ctrl + /
- 여러줄 주석 : /* */
- 자동 완성 : ctrl + space
